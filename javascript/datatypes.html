Data types in JavaScript

there are 2 data types in JavaScript: Primitive and Non-Primitive

primitive data types are 7 and 5 non primitive data types

  1. Number
    2. eg. typeof 7 = number
  2. String
    3. eg. typeof "Hello World!" = string
  3. Boolean
    4. eg. typeof true = boolean
  4. Undefined
    5. eg. typeof undefined = undefined
  5. Null
    6. eg. typeof null = object
  6. BigInt
    7. eg. typeof 9007199254740991n = bigint
  7. Symbol
    8. eg. typeof Symbol() = symbol

we can check the data type of a variable using the "typeof" operator

Boolean

Undefined

Undefined data type has only one value undefined in this case user not assigned any value to the variable it is given by the JavaScript engine

Null

Null data type has only one value null. in this case, user gives a value of null to the variable

BigInt

BigInt data type is used to store integers that are too large to be represented by the Number data type

Symbol

Symbol data type is used to create unique identifiers
