there are mainly 3 types of dialogue boxes

  1. Alert box

    2. Alert box is used to display a message to the user. it is a modal window that pops up on the screen and requires the user to click "OK" to close it.

    ex. alert("Hello, World!");

  2. Confirm box

    3. Confirm box is used to ask the user for confirmation. it is a modal window that pops up on the screen and requires the user to click "OK" or "Cancel" to close it.

    ex. if (confirm("Are you sure?")) { // User clicked "OK" } else { //User clicked "Cancel"}

  3. Prompt box

    4. Prompt box is used to ask the user for input. it is a modal window that pops up on the screen and requires the user to enter some text and click "OK" or "Cancel" to close it.

    ex. let name = prompt("What is your name?");
